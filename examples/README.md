# Quality Examples

This directory contains fake example data for quality validation.

## Landing

- `stations-metadata.json`: Fake stations metadata.
- `variables-metadata.json`: Fake variables metadata.
- `measured-variable.json`: Fake measured variable data.

## Bronze

- `stations-metadata.jsonl`: Fake bronze records for stations.
- `variables-metadata.jsonl`: Fake bronze records for variables.
- `measured-variable.jsonl`: Fake bronze records for measurements.
